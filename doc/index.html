Documentation of ZephyrJS

Zephyr is designed to extend the functionality of PixiJS, rather than wrapping it so that you can always use Pixi without Zephyr.

The Return line below the function indicates the object type that is returned.

PIXI.zephyr

PIXI.zephyr.useKeyInput()
Sets up the key tracking system for use
PIXI.zephyr.useMouseInput()
Sets up the mouse tracking system for use
PIXI.zephyr.useAudio()
Sets up the WebAudio API for Pixi-style usage

PIXI.getMouseFired(mouseButton): Returns boolean - Returns true if this is the first time it has been called since the button has been pressed. - NOTE: Mouse buttons are integers 0 (primary, typically left) 1 (middle), and 2 (secondary, typically right).

PIXI.getMouseDown(mouseButton): Returns boolean - Returns true if the button is actively pressed down. - NOTE: Mouse buttons are integers 0 (primary, typically left) 1 (middle), and 2 (secondary, typically right).

PIXI.getMouseX(): Returns number - Returns the last seen mouse X coordinate, where 0.0-1.0 is within the mouseContainer horizontally from left to right.

PIXI.getMouseY(): Returns number - Returns the last seen mouse Y coordinate, where 0.0-1.0 is within the mouseContainer vertically from top to bottom.

PIXI.collision

PIXI.collision.aabb(a, b): Returns boolean - Checks if the two provided objects are colliding with the aabb algorithm - NOTE: Both a and b are formatted as a javascript object with x, y, width, and height where (x,y) is the top-leftmost point.

PIXI.collision.radius(a, b): Returns boolean - Checks if the two provided objects are colliding with the standard point-radius method of circle collision - NOTE: Both a and b are formatted as a javascript object with x, y, and r where (x,y) is the centermost point and r is the radius.
